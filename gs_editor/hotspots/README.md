# 热点功能使用说明

## 功能概述

热点功能允许用户在PLY模型中添加交互式的3D标记点，可以实现：
- 在3D空间中标记重要位置
- 添加描述信息
- 设置PLY之间的跳转链接
- 保存和加载热点配置

## 使用方法

### 1. 添加热点
1. 选择一个PLY模型
2. 点击热点面板中的"+ 添加热点"按钮
3. 在弹出的对话框中填写：
   - **名称**: 热点的显示名称
   - **位置**: 3D坐标 (X, Y, Z)，或点击"获取当前相机位置"
   - **大小**: 热点的显示大小
   - **描述**: 详细描述信息
   - **目标PLY**: 可选择跳转的目标PLY文件

### 2. 编辑热点
- 点击热点列表中的"✏️"按钮编辑热点
- 点击"🗑️"按钮删除热点
- 点击"🔗"或"📍"按钮跳转或移动相机

### 3. 保存和加载
- **保存**: 点击"💾 保存"按钮导出当前PLY的热点配置为JSON文件
- **加载**: 点击"📁 加载"按钮从JSON文件导入热点配置

## 文件格式

热点配置使用JSON格式：

```json
{
  "version": 1,
  "splatName": "模型名称",
  "timestamp": "2024-01-01T00:00:00.000Z",
  "hotspots": [
    {
      "id": "hotspot_001",
      "name": "热点名称",
      "position": [x, y, z],
      "size": 1.0,
      "description": "描述信息",
      "targetSplatName": "目标PLY名称",
      "visible": true
    }
  ]
}
```

## 功能特性

- ✅ 每个PLY独立的热点系统
- ✅ 3D可视化显示（红色球体+脉冲动画）
- ✅ 点击跳转功能
- ✅ 获取当前相机位置
- ✅ JSON格式导入导出
- ✅ 项目文件中自动保存
- ✅ 实时编辑和预览

## 注意事项

1. 热点数据与特定的PLY文件绑定
2. 跳转功能需要目标PLY已经加载到项目中
3. 保存的JSON文件可以在其他项目中重复使用
4. 热点在3D场景中会随相机距离调整显示大小
