热点系统完整功能测试

🎯 热点基础功能

✅ 取消大小限制：支持任意大小热点
• 在菜单中选择"添加热点"，输入任意大小值（如0.01或100）
✅ 3D悬浮文字：热点名称在3D空间中显示
• 热点上方显示固定14px文字，自动跟随热点位置
✅ PLY绑定显示：热点与PLY同步显示/隐藏
• 切换PLY可见性时，热点自动同步

⚙️ 热点调整功能

✅ 编辑热点属性
• 菜单 -> "编辑热点" -> 修改名称、位置、大小
✅ 设置跳转目标
• 菜单 -> "设置跳转目标" -> 选择热点 -> 选择目标PLY
✅ 快速调整面板
• 右键点击任意热点 -> 打开详细调整面板
✅ 删除热点
• 菜单 -> "删除热点" 或在调整面板中点击删除

🔄 跳转功能

✅ PLY间跳转
• 左键点击热点 -> 自动切换到目标PLY
✅ 跳转状态提示
• 成功跳转时显示确认消息，失败时显示错误信息

💾 数据管理

✅ 保存热点配置
• 菜单 -> "保存热点" -> 下载JSON文件
✅ 加载热点配置
• 菜单 -> "加载热点" -> 选择JSON文件
✅ 项目集成
• 热点数据自动保存到项目文件中

🎮 交互说明

🖱️ 左键点击热点：执行跳转
🖱️ 右键点击热点：打开调整面板
⌨️ 菜单操作：所有基本的增删改查功能

📝 测试步骤

  1. 加载多个PLY文件
  2. 选择第一个PLY，添加热点
  3. 设置热点跳转到第二个PLY
  4. 测试左键点击跳转
  5. 测试右键调整面板
  6. 验证PLY显示/隐藏同步
  7. 测试保存/加载功能
